# Stromworks Train Microcontrollers by Nona Takahara

このリポジトリは、あまりにも複雑になってしまったStormworks向け鉄道車両マイコンを、AI&human readableな文書群に起こし直すことで何とか保守してみようという試みをしているリポジトリです。

作業にはVibe Codingで作ったツール[storm-mcl](https://github.com/nona-takahara/storm-microcontroller-language)が必要ですが、公開までしばらく待ってください。

実機投入用Luaを書く際の共通コーディング規約（8192文字制限との付き合い方など）は [`LUA_CODING_GUIDE.md`](./LUA_CODING_GUIDE.md) を参照。
